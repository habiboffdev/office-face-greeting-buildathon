# Office Face Greeting Buildathon

Welcome to the buildathon.

Your task is to build an office entrance system where:

- videos play continuously in fullscreen
- camera detects known people
- greeting overlay appears without stopping the video
- unknown people do not trigger anything

## Submission Format

Each participant must submit their solution inside:

```text
submissions/your-name/
```

Example:

```text
submissions/ali-valiyev/
```

## Deadline

Final submission: To be announced by the organizers.

## Required Materials

Inside your folder, include:

- working source code
- README.md with setup instructions
- demo video link, 1-3 minutes
- short technical notes

## How to Submit

1. Fork this repository.
2. Clone your fork.
3. Create your folder inside `submissions/your-name/`.
4. Put your full project inside that folder.
5. Add setup instructions in `README.md`.
6. Add your demo video link in `demo-link.txt`.
7. Commit and push your changes.
8. Open a pull request back to this repository.

Pull request title format:

```text
Submission: Your Name
```
