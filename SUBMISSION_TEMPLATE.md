# Submission Template

Create a folder:

```text
submissions/your-name/
```

Recommended structure:

```text
submissions/your-name/
├── README.md
├── src/
├── requirements.txt or package.json
├── demo-link.txt
└── technical-notes.md
```

## README.md

Include:

- project overview
- setup steps
- how to run the app
- required environment variables, if any
- known limitations

## demo-link.txt

Paste one link to a 1-3 minute demo video.

## technical-notes.md

Briefly explain:

- tech stack
- face detection approach
- how known people are stored
- how greeting overlay works
- what you would improve with more time
